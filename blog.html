Blog Post 4: assignment 5

3 Dec 2020

This assignment was pretty fun! It was complex while also relatively simple. These tools, handlebars specifically, felt pretty powerful to use. It was also very interesting to learn how actual websites are hosted server-side. The additional lecture videos were by far the most helpful resource during this assignment. Although, I also used the handlebars website itself for the final part, which was the most difficult part of the assignment as well. Specifically, I'm talking about displaying only one post using the same template as the main site and hiding the other portions of the UI on that page. I had forgot about using the "[ ]" braces when attempting to select only one post from the postData.json file. It didn't hit me until I console.log'd out the individual post data that I was attempting to push through the template, and compared it to the postData.json file. I also struggled with discovering how to use the {{#if }} variable. And after I found that, I forgot the "," when passing that variable to the template, which wouldn't allow my server to start. But overall, it went pretty good!
